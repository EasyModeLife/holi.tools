# Holi QR v0.1.1

A professional, privacy-focused QR code generator powered by Rust and WebAssembly.

## Features
- **Client-Side Generation**: Zero data leaves your device.
- **High Performance**: Optimized WASM engine (~30kb) for instant generation.
- **Customization**: Support for themes, custom shapes, and logos.
- **Vector Export**: High-quality SVG output.

## Tech Stack
- **Core**: Rust (wasm-qr-svg).
- **Frontend**: Astro + Preact.
- **Styling**: Tailwind CSS + @holi/ui.
