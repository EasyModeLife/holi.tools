# About Typst Live

**Typst Live** is a real-time editor for the Typst typesetting language, brought to the web via Holi.tools.

## Purpose
- Provide a responsive, browser-based environment for Typst document creation.
- leverage WebAssembly for high-performance rendering.

## Tech Stack
- **Framework:** Astro
- **Interactivity:** React + WASM (Planned)
- **Styling:** Tailwind CSS
